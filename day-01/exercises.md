## 💻 Exercises - Day 1

### Level 1

1. Check the python version you are using
2. Open the python interactive shell and do the following operations. The operands are 3 and 4.
   - addition(+)
   - subtraction(-)
   - multiplication(*)
   - modulus(%)
   - division(/)
   - exponential(**)
   - floor division(//)

### Level 2

1. Write a comment saying python is easy to learn
2. Declare variables of different data types: string, integer, float, complex, boolean
   - name: string
   - age: integer
   - height: float
   - complex_number: complex
   - is_married: boolean
   - is_true: boolean
   - is_light_on: boolean
3. Declare a variable with multiple values on one line: first_name, last_name, country, age, is_married, is_light_on

### Level 3

1. Find the length of your first name and last name using len() method
2. Compare the length of your first name and your last name
3. Declare two variables called first_name and last_name, assign your first and last name. Declare a variable called full_name and assign the value of first_name and last_name separated by space. Declare a variable called country and assign your country name. Declare a variable called city and assign your city name. Declare a variable called age and assign your age. Declare a variable called year and assign the current year. Declare a variable called is_married and assign a boolean value. Declare a variable called is_true and assign a boolean value. Declare a variable called is_light_on and assign a boolean value. Declare multiple variable on one line: first_name, last_name, country, age, is_married, is_light_on, city, year
4. Use the built-in function type() to check the data type of the different variables you created.
